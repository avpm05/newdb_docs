# Юридические лица

Раздел содержит методы NEWDB для проверки юридических лиц по ИНН: арбитражные дела, блокировки счетов, признаки банкротства, исполнительные производства ФССП и сведения из ЕГРЮЛ.

## Доступные методы

- [arbitr_legal](01-arbitr_legal.md) — арбитражные дела юрлица в КАД
- [fns_block](02-fns_block.md) — блокировки счетов юрлица по данным ФНС
- [bankrot_legal](03-bankrot_legal.md) — банкротство юрлица через Федресурс
- [egrul](04-egrul.md) — сведения ЕГРЮЛ и карточка компании из Прозрачного бизнеса
- [fssp_legal](05-fssp_legal.md) — исполнительные производства ФССП по ИНН юридического лица
