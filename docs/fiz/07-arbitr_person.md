# arbitr_person — Проверка арбитражных дел (физлица, КАД)

POST `https://api.newdb.net/v2`

Метод обеспечивает поиск и анализ зарегистрированных в системе КАД (kad.arbitr.ru) арбитражных дел участия физического лица (идентификация по ИНН), со сведениями о процессуальном статусе, составе участников, комплектах судебных актов и хронологии судебных действий. На основе анализа судебной документации в формате PDF выделяются денежные требования и санкции, определяется суть споров, выполняется классификация дела по категориям (договорные споры, банкротство, имущественные претензии, налоговые доначисления и др.), а также выделяется применимое законодательство.

---

## Заголовки

Content-Type: application/json  
X-API-KEY: <your_token>

---

## Входная схема (request)

```json
{
  "params": {
    "innfiz": "string (ИНН физлица)",
    "country": "string (ru)",
    "method": "arbitr_person"
  },
  "webhook": "https://your.host/whook",
  "requestId": "optional-string"
}
```

---

## Пример запроса

```json
POST /v2 HTTP/1.1
Host: api.newdb.net
Content-Type: application/json
X-API-KEY: YOUR_TOKEN

{
  "params": {
    "innfiz": "644605034188",
    "country": "ru",
    "method": "arbitr_person"
  },
  "requestId": "a5962f88-2916-4179-b59d-43c123faa911"
}
```
