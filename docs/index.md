# NEWDB — REST API

Добро пожаловать в документацию NEWDB. Здесь вы найдёте описание REST API, примеры запросов/ответов, а также **метаданные для AI-агентов** (раздел `x-ai` в описаниях методов).

## Базовый URL

```
https://api.newdb.net/v2
```

## Аутентификация

Передавайте токен в заголовке:

```
X-API-KEY: <your_token>
```

API разработан для проведения комплексных проверок юридических и физических лиц. Для получения токена напишите письмо на access@newdb.net.

## Общий формат запроса

Отправьте POST на `https://api.newdb.net/v2`:

```json
{
  "params": { "...": "..." },
  "requestId": "<uuid4>",
  "webhook": "<url>"
}
```

| Параметр            | Обязательный | Описание                                                                                               |
|---------------------|--------------|--------------------------------------------------------------------------------------------------------|
| **params**          | Да           | Параметры запроса                                                                                      |
| **params.country**  | Да           | Код страны (2 буквы) — ISO 3166-1 alpha-2                                                              |
| **params.method**   | Да           | Метод API                                                                                              |
| **requestId**       | Нет          | UUIDv4 идентификатор запроса. При повторном использовании API вернёт старый результат без нового запроса |
| **webhook**         | Нет          | URL для уведомлений об обновлениях по запросу                                                          |

## Методы по физическим лицам

| Метод                    | Описание                                                                |
|--------------------------|-------------------------------------------------------------------------|
| **complex_by_passport**  | Комплексная проверка физлица по паспорту и всем источникам              |
| **passport_fns**         | Проверка паспорта через ФНС + получение ИНН                             |
| **passport_mvd**         | Проверка паспорта по реестру недействительных паспортов                 |
| **arbitr_person**        | Арбитражные дела физлиц в КАД (по ИНН)                                  |
| **fssp_person**          | Исполнительные производства ФССП                                        |
| **fedresurs_bankrot**    | Наличие сообщений о банкротстве (ЕФРСБ)                                 |
| **pledge_person**        | Залоги/обременения физлиц (ФНП + Федресурс)                             |

## Методы по юридическим лицам

| Метод             | Описание                                                |
|-------------------|---------------------------------------------------------|
| **arbitr_legal**  | Арбитражные дела юрлиц в КАД (по ИНН)                   |

## Методы по имуществу

| Метод               | Описание                                                                                  |
|---------------------|-------------------------------------------------------------------------------------------|
| **rosreestr**       | Справочная информация по объектам недвижимости (онлайн, Росреестр)                       |
| **pledge_property** | Проверка залогов/обременений по ID предмета (ФНП + Федресурс)                             |
| **pledge_vin**      | Проверка залогов/обременений транспорта по VIN (ФНП + Федресурс)                          |
