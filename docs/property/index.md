# Имущество

Раздел содержит методы NEWDB для проверки объектов недвижимости, транспортных средств, залогов и обременений.

## Доступные методы

- [rosreestr](01-rosreestr.md) — проверка объекта недвижимости по кадастровому номеру или адресу
- [pledge_property](02-pledge_property.md) — залог и обременения по идентификатору предмета
- [pledge_vin](03-pledge_vin.md) — залог и обременения по VIN
- [nspd_cadastr](04-nspd_cadastr.md) — получение геоданных и характеристик объекта по кадастровому номеру
